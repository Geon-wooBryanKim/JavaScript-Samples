# JavaScript Examples

### Usage
   - node 와 함께 동작
   
       `node SOURCE-FILE.JS`

## 01.Data : 데이터 다루기

* 변수와 상수
* 데이터 타입
* 주요 데이타 타입
* 밸류 타입과 레퍼런스 타입

## 02.Operator
   * equal : 동등 비교
   * in_delete : in, delete 연산자
   * instanceof : instanceof 연산자
   * and_or : and, or 연산자

## 03.Control
   * forLoop : for 반복문

## 04.Function
   * function : 함수 선언과 호출
   * parameter : 파라미터 
   * overloading : 함수 오버로딩
   * variadic_parameters : 가변길이 파라미터
   * innser_function : 함수 내 함수
   * iife : Immediately Invoked Function Expression

## 05.Closure
   * closure : 클로저, 함수 객체
   * closure_context : 클로저 콘텍스트
   * closure_parameter : 파라메터로서 클로저   

## 06.Class
   * object : 객체 리터럴, 프로퍼티와 메소드   
   * constructor : 생성자 함수를 이용한 객체 생성과 사용
   * property : 프로퍼티
   * class : class ( ES6 )

## 07.ErrorHandling
   * try_catch : 에러 처리
   * throw : 에러 발생   

## 08.ADV
   * var : JavaScript 변수의 특징
   * global_variable : 글로벌 변수
   * strict_mode : strict mode
   * scope : 변수 스코프
   * let : 블록 스코프 변수 let. (ES6)
